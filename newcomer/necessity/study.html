                                            # 推荐在学校超市直接购买的

                                            提示

                                            标记 “*” 的意味着往年校内超市有售

                                            # 笔*

                                            大部分同学高中应该都养成了用黑色笔的习惯。大学文科专业可能需要记录的东西有不少。一些理科专业譬如物理也建议做好笔记。笔没有高中那么费。有条件 (有自制力) 的人用 iPad 做笔记也是可以的。

                                            iPad

                                            买来追剧不学习的人有不少，请自行评估自制力。

                                            # 本*

                                            建议使用活页夹。

                                            大学跟高中不太一样，每一门课都是一本书，一些课上记的东西课下可能还要誊抄。所以强烈推荐买一个那种活页的本子，这样可以很方便的将一些不需要的页去掉或添加进新页。(这也正是知乎各答主和很多学长学姐推荐大家使用 iPad 的原因，因为 iPad 在做笔记的时候可以很容易的去涂抹修改自己做的笔记)

                                            # 订书机与订书针*

                                            重要，一些课程复印的材料，学长学姐笔记的复印件，甚至一些自己要看的论文，交的作业和材料都需要装订。

                                            可以不急着买，刚开学问问哪个室友有借来用就行了。

                                            除了考虑自备小订书机外，Mr.Hope 强烈推荐整个寝室一起出钱网购一个大的，在装订厚材料的时候用 (相信我您会遇到数次正常尺寸订书机很难订上的材料的)，这样大家都不吃亏。

                                            # 建议网购

                                            # 小台灯*

                                            充电式台灯可以在宿舍熄灯后用，但光亮持续的时间短；宿舍有电时用带插头的台灯更好。

                                            Mr.Hope 的建议是不用一定买内置电源的台灯，买 USB 接口的即可，熄灯后接上充电宝用。推荐的理由是台灯功率不小，电池损耗会很快，内置电源的台灯大三大四一停电就很快趴窝，到时候可能还要换新的。

                                            建议网购的原因是学校往年卖的质量也就一般，性价比肯定是不如淘宝爆款的。

                                            # 手表

                                            体测长跑计时，期末考试时看时间，还是会用到的。

                                            # 书包

                                            建议自己提前购买，买一个有点个性的免得上下课、在图书馆的时候别人错拿了。

                                            严重警告

                                            不要淘宝上随便买!!! 不要买热销爆款!!! 购买时请仔细挑选款式!!!

                                            Mr.Hope 本科时附近三个男生寝室 16 个人有 6 个人用同款书包。书包撞款不仅尬，还极其容易被错拿。

                                            # U 盘

                                            强烈建议买两个，一款便宜的专门拷东西用，另一个选质量好一些容量大一些的存放一些重要文件的备份。

                                            小心计算机病毒

                                            很多人 (可能就是您) 压根不懂电脑，可能会时常用自己的 U 盘感染或被感染学院或打印社的电脑。

                                            请一定分开您拷东西的 U 盘和备份的 U 盘，以避免自己电脑文件和备份文件同时损坏。

                                            # 工具书

                                            除了对应专业的参考书籍之外，大家可能还需要教师资格证、四六级、计算机等级考试等相关工具书。

                                            # 文具袋*

                                            大学大部分一根笔就解决了 (除非您有给书或者笔记用各种颜色的笔做标记的情况，或是美术专业)。

                                            # 随身带的小笔记本*

                                            • 一些人的观点是非常有用。学校平时一些零碎的通知会很多，而且自己可能有一些事情或者是问题也可以记在本子上，尤其是开学初期。

                                            • Mr.Hope 的观点是手机备忘录、待办事项提醒更好用，到时间还会有提醒。
