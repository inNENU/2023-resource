                                                            # 概况

                                                            • 寝室楼内每层均设有自习室。

                                                            • 自习室不间断供电。

                                                            • 请不要将个人物品堆放到自习室内，宿舍保洁阿姨会定期清理，如果清理时没人认领，自动当作废弃物处理。

                                                            # 自习室照片

                                                            # 自习室使用公约

                                                            自习室是自主学习的公共场所，为维护一个秩序良好文明健康的学习环境，确保自习室的有效使用，特制定以下公约，请同学们共同自觉遵守:

                                                            1. 自习室座位随来随用，请勿占座，工作人员会定期将占

                                                            2. 品集中存放于书柜处，若有遗失，后果自负;

                                                            3. 保持自习室良好秩序，进入自习室后请自觉将电子产品

                                                            4. 静音模式，请勿在自习室内接打电话、高声讨论;

                                                            5. 请自觉爱护公共设施，请勿在桌椅、门窗、墙壁上乱涂或张贴各种广告;

                                                            6. 请自觉注意个人卫生，及时清理垃圾，主动承担座位附近卫生工作;

                                                            7. 请勿在自习室内吸烟、酗酒;

                                                            8. 如发现任何问题隐患，请及时告知工作人员

                                                            # 温馨提示

                                                            自习室清理时间为:

                                                            1. 上午08:00-09:00

                                                            2. 中午12:00-13:00

                                                            3. 晚上17:00-18:00

                                                            学生宿舍自我管理委员会

                                                            2019年11月

                                                            # Mr.Hope の 吐槽

                                                            对于自习室来说，自习室整夜都有电照明，晚上学习不想让灯光打扰到已经休息的室友的话，最好带着东西去自习室学。

                                                            在女寝这边，由于女生东西比较多，可能会有个别女生把东西都堆在自习室，凭借阿姨不好意思真的都给扔了霸占自习室很多座位。这种浪费公共资源的行为，如果大家遇到了，可以积极给学生处打电话反映情况。
